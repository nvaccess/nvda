NonVisual Desktop Access (NVDA)
網址: http://www.nvda-project.org/

NonVisual Desktop Access (NVDA) 是一套免費且開放源代碼在微軟視窗系統操作的螢幕報讀軟體, 它可讓全失明或視力受損人士操作視窗系統而無須依賴健視者的協助. NVDA 可從操作系統中抓取畫面上的資料, 然後透過語音合成器朗讀出來, 所以用家能掌握顯示器畫面上的情況.

要獲取更多的資訊, 可參考程式的用戶指南. 此外, 程式的網站也具備更多相關的資源和資訊.
