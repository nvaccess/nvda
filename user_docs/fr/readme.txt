NonVisual Desktop Access (NVDA)
URL: http://www.nvda-project.org/

NonVisual desktop access (NVDA) est un lecteur d'écran open-source et gratuit
pour le système d'exploitation Microsoft Windows.
il permet aux personnes aveugles ou malvoyantes d'utiliser un ordinateur
sous Windows sans coût additionnel par rapport à une personne voyante.
NVDA permet à l'utilisateur de savoir ce qui se passe à l'écran
en interrogeant le système et en restituant l'information
au moyen d'une synthèse vocale.

Pour plus d'informations, veuillez consulter le guide de l'utilisateur
inclus dans le logiciel.
Des informations supplémentaires sont également disponibles sur le site
web de NVDA.
